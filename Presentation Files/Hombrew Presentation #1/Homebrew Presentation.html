Perliminary Analysis

So, from here on we will talk about the nitty gritty of how the Analysis was done. Many numbers follow, but not too many, don't worry.

The Process

So, what did I actually do?
The analysis had 3 main stages (some with sub stages):

  1. Data processing and scoring: The code processes the survey data to calculate scores for each homebrew idea based on the specified features. It iterates through each "feature" in ['Intresting', 'Special / Original', 'Fun', 'Include in our Game', 'Depth'] and performs the following steps:
    1. If the current feature is 'Fun' or 'Include in our Game', it checks for really low scores (1 or 2) and prints a message about the number of such cases.
    2. Calculates the average and Basically, how different were the answers from each other standard deviation for each idea's ratings on the current feature and computes a average - standard deviation, and a normalization to make it a percentage normalized score.
    3. Creates two bar charts using plotly: one for the average rating with error bars and the other for the normalized score. The charts are saved and will be shown below.
  2. Checking multiple scoring formulas: The code defines four different scoring formulas (formula_A, formula_B, formula_C, and formula_D). These formulas assign weights to each feature (Include, Fun, Depth, Intresting, and Special) to calculate a final score for each homebrew idea.
  3. Final scoring and visualization:The code iterates through each scoring formula and calculates the final score for each homebrew idea using that formula. It then creates a bar chart using plotly to display the final scores for each idea based on the current formula. These are the results shown above.

Given more time, I would have given some more weight to these specific low scores in the analysis - if its extremely not fun, or extremely not something you would want in the game, I want the results to mirror that, since those are 2 really important factors.
As it is, I think the results reflect these sentiments anyway, so no harm done.

But let's talk about what I can learn from these really low scores, as it can be interesting:

  • Effect of Aging seems to be very poorly recived on both accounts. I would wager this has to do with some problem in how I presented it, since the 1st edition rules provided as an example seem very harsh and unforgiving (as was 1st edition).
    As mentioned in the first part, I think they need a substantial amount of work to make them work, but these results also emphasize that there need to be some extra care taken to make sure they are presented in a way that will seem appealing.
  • Path of the Spellshield gets a low score, and I think the main issue with it is that its role and lore aren't very clear. The idea of an anti spellcaster warrior is very well liked (just look at the result of oath of inquisition), so the main takeaway here is that more explanations were needed to fully portray the fantasy this subclass embodies.
  • The last interesting result is low fun score for Multiclassing. I think this makes perfect sense - going to the doctor isn't fun, but necessary. This is something that the game needs, but giving it a low fun score makes sense.
